# Smalltalk

- Platform-independent virtual machine
- Garbage collector
- JIT compilation
- Everything is an object
- Everything happens by sending messages
- Dynamically typed
- State is private to objects
- Methods are public
- Every object is a reference
- Unused objects are garbage-collected
- Single inheritance
